# runesleo/claude-code-workflow

## What It Does
Production-grade workflow template for Claude Code with sophisticated memory management. Implements a 3-layer context system (global rules, project docs, hot context) with automatic loading and verification.

## Key Features
- **3-Layer Memory**: Global rules + project docs + hot context
- **SSOT Framework**: Single source of truth for project knowledge
- **Auto-Save/Resume**: Session persistence across interruptions
- **Verification Gates**: Built-in review and testing checkpoints
- **Task Routing**: Intelligent routing to appropriate Claude capabilities

## Productivity Impact
- Battle-tested patterns from production use
- Reduces context setup time for new tasks
- Prevents context loss on interruptions
- Verification system catches errors early

## Relevance: HIGH

Eric should adopt these patterns. The 3-layer context management is particularly valuable for complex, long-running projects.

## Potential Integration
- Adopt SSOT framework for MEMORY.md
- Implement 3-layer context loading
- Add verification gates to critical workflows

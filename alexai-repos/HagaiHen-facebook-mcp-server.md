# HagaiHen/facebook-mcp-server

## What It Does
MCP (Model Context Protocol) server for Facebook Page automation. Exposes Facebook Graph API operations as LLM-callable tools including posting, comment moderation, analytics, and messaging.

## Key Features
- **Post Management**: Create, update, schedule posts
- **Comment Moderation**: Reply, hide, delete, filter negative comments
- **Analytics**: Impressions, engagement, reactions, shares
- **DM Support**: Send direct messages to users
- **Claude Integration**: Ready-to-use with Claude Desktop

## Productivity Impact
- Automate Facebook Page management
- Auto-post daily digests to Facebook
- Moderate comments without manual work
- Track engagement metrics programmatically

## Relevance: MEDIUM

Could automate posting of repo digests and other updates to AlexAI Facebook page. Reduces manual social media work.

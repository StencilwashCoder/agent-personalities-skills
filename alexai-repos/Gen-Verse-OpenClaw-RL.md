# Gen-Verse/OpenClaw-RL

## What It Does
Reinforcement Learning framework for training personalized AI agents from natural conversation feedback. Supports Binary RL, OPD (Online Preference Distillation), and LoRA fine-tuning. One-line deployment.

## Key Features
- **Conversation RL**: Learn from natural chat feedback
- **Binary RL**: Simple thumbs up/down training
- **OPD Support**: Online preference distillation
- **LoRA Fine-tuning**: Efficient parameter updates
- **One-Line Deploy**: `pip install openclaw-rl && openclaw-rl train`

## Productivity Impact
- Personalized agents that learn your preferences
- No need for complex training pipelines
- Continuous improvement from usage
- Customized responses based on feedback history

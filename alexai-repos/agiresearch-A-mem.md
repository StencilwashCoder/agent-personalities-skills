# agiresearch/A-mem

## What It Does
Agentic memory system for LLM agents implementing Zettelkasten principles. Features dynamic linking between memories, hierarchical organization ( fleeting → literature → permanent ), and automatic memory evolution based on access patterns and relevance.

## Key Features
- **Zettelkasten-inspired**: Atomic notes with unique IDs and bidirectional linking
- **Dynamic Evolution**: Memories upgrade/downgrade based on importance and retrieval
- **Contextual Retrieval**: Semantic search with temporal and associative context
- **Agent-native**: Designed specifically for autonomous agent workflows

## Productivity Impact
- Reduces redundant API calls by maintaining richer context
- Enables agents to build knowledge over long-term interactions
- Better recall of relevant information from past conversations
- Self-organizing memory reduces manual curation

## Relevance: HIGH

Could significantly improve my memory system beyond the current simple file-based approach. The dynamic linking and evolution concepts are particularly interesting for long-term knowledge retention. Worth testing against current memory implementation.

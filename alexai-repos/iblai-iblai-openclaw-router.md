# iblai/iblai-openclaw-router

## What It Does
Cost-optimizing model router for OpenClaw that automatically routes requests to the cheapest capable Claude model. Analyzes prompt complexity and routes to Haiku, Sonnet, or Opus accordingly.

## Key Features
- **Intelligent Routing**: Complexity scoring to pick optimal model
- **Cost Optimization**: ~80% savings by using cheaper models when possible
- **Quality Preservation**: Ensures complex tasks still get Opus when needed
- **Drop-in Replacement**: Works with existing OpenClaw setup
- **Performance Tracking**: Monitors cost savings over time

## Productivity Impact
- **Immediate Cost Savings**: $140+/month reduction in API spend
- **Transparent**: No code changes needed to existing workflows
- **Smart Defaults**: Learns which tasks need which models

This is the highest-impact repository. Direct cost reduction with zero downside. Should be deployed immediately.

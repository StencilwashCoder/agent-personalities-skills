# Intrect-io/OpenSwarm

## What It Does
Autonomous AI agent orchestrator powered by Claude Code CLI. Orchestrates multiple Claude instances as autonomous agents with Worker/Reviewer pair pipelines, Linear integration, Discord control, and long-term memory via LanceDB.

## Key Features
- **Multi-Agent**: Worker/Reviewer/Tester/Documenter pipeline
- **Linear Integration**: Auto-fetch and process Linear issues
- **Discord Control**: Full command interface for monitoring
- **Memory**: LanceDB vector embeddings for long-term recall
- **Web Dashboard**: Real-time monitoring on port 3847
- **PR Auto-Improvement**: Auto-fix CI failures and conflicts

## Productivity Impact
- Automates complex coding workflows
- Multi-agent collaboration for higher quality
- Persistent memory across sessions
- Discord control for remote management

## Relevance: MEDIUM-HIGH

Useful for complex projects requiring multiple specialized agents. The Worker/Reviewer pattern is particularly valuable for code quality.

🚀 Just shipped 5 new open source tools in one week.

Built a fleet of Model Context Protocol (MCP) servers that let AI assistants like Claude control everything from Bitcoin nodes to Proxmox VMs through natural language.

What's MCP? Think "USB for AI tools" — one standard protocol, endless integrations.

**The stack:**
📊 mcp-civic-data — 100+ government APIs (FEMA, weather, congress)
₿ mcp-bitcoin-cli — Bitcoin Core RPC interface
🖥️ mcp-proxmox-admin — VM/container management
📦 mcp-ipfs — Decentralized storage integration
🔐 mcp-multi-agent-ssh — Fleet orchestration

All include:
✅ TypeScript + strict type checking
✅ Comprehensive test suites
✅ CI/CD pipelines
✅ Docker support
✅ MIT license

The goal: make infrastructure management as simple as having a conversation.

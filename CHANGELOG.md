# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-26

### Added
- Initial release of MCP Server Validator
- JSON Schema validation for MCP configurations
- Support for stdio, HTTP, and SSE transports
- Tool definition validation with duplicate detection
- Security auditing for auth configuration
- Custom rules system for organization-specific requirements
- Multiple output formats: pretty, JSON, and SARIF
- GitHub Security tab integration via SARIF
- Full test suite with Jest
